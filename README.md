# Mlog

This is the implementation of the paper “MLog: Towards Declarative In-Database Machine Learning”. 

## Dependencies:
	jdk 1.8
	maven
	libzmq
	libjzmq

## Usage:
	compile project:	./compile.sh
	clear:				./clear.sh
	run program:		./run.sh <file path>

## Example:
	./compile.sh
	./run.sh imdb.bl
